A simple tool to fix typos from a list of suspect words on your website

== Description ==
The tool asks to upload a list, producing a list is done by a crawler that scans given link and its included links. The output is csv file in the format: typo, correction, link to post, context to search, longer context to preview (comma separated). Then it shows the list with correct+dismiss buttons, correct actually edits so you don\'t need to edit each post and locate the typo in it and then save, one click and the typo is corrected.
